Level 5: Microservice化
=============================================================

目的・ゴール:　Microserviceを支えるインフラの技術・テクノロジースタックを理解する
=============================================================

Level4までで既存のアプリケーションをコンテナ化して本番運用可能かつ迅速にリリースできる仕組みを作りました。
このレベルではアプリケーションをコンポーネント単位に分割していった際に必要となるインフラの仕組みを適用します。
